# External Dependencies

These are the primary upstreams referenced in the flake inputs. Use them in specific modules or suites.

- `nixpkgs` - NixOS unstable packages (nixos-unstable channel).
- `home-manager` - User environment management.
- `stylix` - System-wide theming and visual configuration.
- `disko` - Declarative disk partitioning for host installs.
- `hyprflake` - Hyprland desktop environment stack.
- `waybar-auto-hide` - Auto-hiding waybar module.
- `llm-agents` - LLM agent tools (numtide/llm-agents.nix).
- `nix-flatpak` - Declarative Flatpak integration.
- `nixos-hardware` - Hardware-specific NixOS modules.
- `spicetify-nix` - Spotify theming via Spicetify.
- `affinity-nix` - Affinity suite (Photo/Designer/Publisher) on Linux.
- `apple-fonts` - Apple font packages (SF Pro, SF Mono, NY).
- `nixos-vscode-server` - VS Code remote server support.
- `determinate` - Determinate Systems Nix distribution.
- `browser-previews` - Browser preview/beta builds.
